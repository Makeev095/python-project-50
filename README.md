# DESCRIPTION:

**Вычислитель отличий**

Запускается из командной строки и вычисляет отличия между двумя файлами. На данный момент работает с JSON и YAML.

**Запуск справки:**

`gendiff -h`

**Запуск скрипта c настройками по-умолчанию:**

`gendiff <file_path1> <file_path2>`
